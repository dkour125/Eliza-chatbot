🧠 ELIZA Project Dashboard

Enhanced Chatbot with Sentiment Analysis & Real-time Analytics

📋 Project Overview

A modern, feature-rich implementation of Joseph Weizenbaum's classic ELIZA chatbot (1966). Built for CS5001 Object-Oriented Modelling, demonstrating advanced software engineering principles with multiple personalities, sentiment analysis, and comprehensive analytics.

🚀 Quick Start Guide

Installation

pip install -r requirements.txt

Run Web Application

python app.py
# Open http://localhost:5000

Run CLI

python eliza.py --personality therapist
python eliza.py --personality child
python eliza.py --personality mentor

Run Analytics Dashboard

python analytics_dashboard.py
